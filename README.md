Big-8 裁員戰情室 2025

這是一個即時監測 2025 年全球八大科技巨頭 (Google, Microsoft, Meta, Amazon, Apple, Nvidia, Broadcom, Tesla) 裁員動態的網頁應用。結合了賽博龐克視覺風格與 Gemini 2.5 AI 自動化分析。

🚀 快速開始 (GitHub Pages)

建立儲存庫：在 GitHub 上建立一個名為 layoff-war-room-2025 的新儲存庫。

上傳檔案：

將 index.html 下載並上傳至儲存庫根目錄。

將本 README.md 上傳至儲存庫。

啟用網頁：

進入儲存庫的 Settings > Pages。

在 Build and deployment > Branch 選擇 main 分支並點擊 Save。

獲取金鑰：

訪問網頁後，點選右上角的 設定 API 圖示。

貼入您的 Google AI Studio API Key 以啟動 AI 搜尋與分析功能。

🛠️ 技術棧

Frontend: Tailwind CSS, FontAwesome 6

AI Engine: Gemini 2.5 Flash (via Google Generative AI API)

Search: Google Search Grounding

Hosting: GitHub Pages
